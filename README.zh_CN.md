## dde-app-services

dde-app-services提供dde-applications的服务集合，包括配置中心。

## 依赖

### 编译依赖

* Qt (>= 5.6)
* [dtk](https://github.com/linuxdeepin/deepin-tool-kit)

## 安装

### 从源代码构建

1. 确保已经安装了所有的编译依赖.

2. 构建:

```bash
mkdir build
cd build
qmake ..
make
```

3. 安装:

```bash
sudo make install
```
